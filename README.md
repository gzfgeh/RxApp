  本App是本人最新学习新知识的一个总结，在工作之余所做，App采用Material Design设计效果，MVP+RxJava+Retrofit+Okhttp架构，由于是学习之用就没有考虑到兼容问题，目前支持Android4.0以上版本，后续会持续更新

            RxAndroid: RxJava的辅助库; <br>
